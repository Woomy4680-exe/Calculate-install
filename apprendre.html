Apprendre à utiliser calculate.

C'est bon votre cl est installée? Nous allons maintenant apprendre a l'utiliser.

Gestion des applications avec portage.

Pour la rechercher:

eix (nom de l'application)

Pour l'installer:

emerge (nom de l'app)

Pour la désinstaller:

emerge -C( majuscule essentielle) (nom de l'app)

Pour la mettre à jour:

emerge -U (nom de l'app)

Exemples:

  • emerge libreoffice
  • emerge -C libreoffice
  • emerge -U sys-kernel/calculate-sources

Mise a jour du système avec les outils calculate.

Lancer la mise a jour du système:

Lancez un terminal en root et tapez : cl-update -o

Attendez, cela prendra un certain temps la première fois.

Je conseille de lancer la mise à jour une fois par semaine.

Une fois la mise à jour terminée vous pouvez redémarrer si néçaissaire ( si vous avez mis à jour le kernel)(si le kernel a été mis à jour, il est préférable de relancer la mise à jour après quelques redémarrage pour supprimer l'ancien noyau)

Changer la version de son kernel vers la dernière avec les package keywords

Si vous souhaitez passer sur le dernier kernel disponible, rien de plus simple:

Lancez un terminal en root et mettez a jour le système. Apres ça tappez la commande suivante: nano /etc/portage/package.keywords/newkernel

Une fois dedans tapez : sys-kernel/calculate-sources

Faites un controle+O sauvegardez et quittez avec control+x

Mettez à jour le système avec cl-update.
