Apprendre à utiliser calculate.

C'est bon votre cl est installée? Nous allons maintenant apprendre a l'utiliser.

Gestion des applications avec portage.

Pour la rechercher:

eix (nom de l'application)

Pour l'installer:

emerge (nom de l'app)

Pour la désinstaller:

emerge -C( majuscule essentielle) (nom de l'app)

Pour la mettre à jour:

emerge -U (nom de l'app)

Exemples:

  • emerge libreoffice
  • emerge -C libreoffice
  • emerge -U sys-apps/flatpak

Mise a jour du système avec les outils calculate.

Lancer la mise a jour du système:

Lancez un terminal en root et tapez : cl-update -o

Attendez, cela prendra un certain temps la première fois.

Je conseille de lancer la mise à jour une fois par semaine.

Une fois la mise à jour terminée vous pouvez redémarrer si néçaissaire ( si vous avez mis à jour le kernel)(si le kernel a été mis à jour, il est préférable de relancer la mise à jour après quelques redémarrage pour supprimer l'ancien noyau)

Changer la version de son kernel vers la dernière avec les package keywords

Si vous souhaitez passer sur le dernier kernel disponible, rien de plus simple:

Lancez un terminal en root et mettez a jour le système. Apres ça tappez la commande suivante: nano /etc/portage/package.keywords/newkernel

Une fois dedans tapez : sys-kernel/calculate-sources

Faites un controle+O sauvegardez et quittez avec control+x

Mettez à jour le système avec cl-update -o.

Rajouter un overlay avec Layman

Si vous souhaitez installer des programmes supplémentaires ou autres, il sera peut être nécaissaire d'installer un overlay même si vous ne souhaitez pas le faire je vous conseille l'adrien overlay qui est très utile .

Pour installer un overlay

curl https://raw.githubusercontent.com/aaaaadrien/adrien-overlay/master/repositories.xml (ou le lien qui vous sera donné)-o /etc/layman/overlays/adrien-overlay.xml ( le nom de l'overlay avant le .xml)

layman -S

layman -a adrien-overlay(ou le nom de l'overlay)

Note pour l'adrien overlay : après l'installation, lancez cette commande : cl-core-variables --set update.cl_update_other_set=on

Voila, l'overlay est maintenant installé ;-}

Rajouter un dépot flatpak pour installer plus d'applications:

La première étape sera d'installer l'adrien-overlay comme marqué au dessus.

Après, vous aurez juste a lancer cette commande : emerge sys-apps/flatpak puis installer le dépot flathub avec cette autre commande: flatpak remote-add --if-not-exists flathub https://flathub.org/repo/flathub.flatpakrepo s et a reboot votre pc.

Gestion des services avec OpenRC.

Etant donné que SystemD n'est pas présent sous calculate linux, tout les services se gèrent avec OpenRC .

Pour activer un service au démarrage:

rc-update add $service default

Ex: rc-update add sshd default

Pour désactiver un service au démarrage:

rc-update del $service default

Ex: rc-update del sshd default

Lancer un service

/etc/init.d/$service start

Ex: /etc/init.d/sshd start

Stopper un service

/etc/init.d/$service stop

Ex:/etc/init.d/sshd stop

Relancer un service:

/etc/init.d/$service restart

Ex: /etc/init.d/sshd restart

Reload un service ( recharger la config sans le stopper)

/etc/init.d/$service reload

Ex: /etc/init.d/sshd reload

Obtenir le statut d'un service:

/etc/init.d/$service status

Ex: /etc/init.d/sshd status
